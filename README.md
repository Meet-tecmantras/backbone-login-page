# Backbone Login Page

A minimal Backbone.js login page demo using vanilla JavaScript modules, Backbone Models/Views, and a polished UI.

## Setup

1. `npm install`
2. `npm run start`

The `start` script runs `live-server` rooted at the `frontend` directory on port `4173`. Open http://127.0.0.1:4173 to interact with the login page.

## Demo Credentials

- **Email:** `user@example.com`
- **Password:** `password123`

Entering the demo credentials will show a success state; other combinations simulate an authentication failure with inline feedback.
